# 波动光学交互式仿真平台

基于Python + Streamlit开发的交互式光学实验仿真平台，支持双缝干涉、单缝衍射、多缝光栅和迈克耳孙干涉等实验的实时仿真。

## 功能特点

- **双缝干涉**: 模拟杨氏双缝干涉实验，实时观察条纹间距变化
- **单缝衍射**: 观察单缝衍射图样及其中央明纹宽度
- **多缝光栅**: 演示多缝光栅衍射，了解光栅常数与分辨率关系
- **迈克耳孙干涉**: 模拟迈克耳孙干涉仪，观察条纹变化

## 两种运行模式

1. **实验模式**: 用于展示光学现象，适合课堂教学演示
2. **练习模式**: 提供练习题，帮助学生自主学习和检验理解

## 快速开始

### 方法一：使用启动脚本（推荐）

双击运行 `run_simulation.bat` 文件

### 方法二：命令行启动

```bash
pip install -r requirements.txt
streamlit run wave_optics_simulation.py
```

然后在浏览器中打开 http://localhost:8501

## 参数说明

### 双缝干涉
- **波长**: 400-800 nm
- **缝距**: 0.1-2.0 mm
- **屏距**: 0.5-3.0 m

### 单缝衍射
- **波长**: 400-800 nm
- **缝宽**: 0.02-0.5 mm
- **屏距**: 0.5-3.0 m

### 多缝光栅
- **波长**: 400-800 nm
- **光栅常数**: 5-50 μm
- **缝数**: 5-20
- **缝宽**: 1-25 μm

### 迈克耳孙干涉
- **波长**: 400-800 nm
- **镜面移动距离**: 0-10 μm

## 物理公式

### 双缝干涉
条纹间距: Δx = λD/d

### 单缝衍射
第一暗纹条件: a·sinθ = λ

### 多缝光栅
光栅方程: d·sinθ = kλ

### 迈克耳孙干涉
条纹变化: ΔN = 2d/λ
